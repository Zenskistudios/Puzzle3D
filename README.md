# Puzzle3D
🎮 Game Description – Sliding Puzzle Challenge

Sliding Puzzle Challenge is a classic 4×4 tile puzzle game that tests your logic, focus, and problem-solving skills. Rearrange the shuffled image tiles to restore the original picture.

🧩 Features:

Adjustable puzzle sizes (e.g., 4×4 and more)

Move and Rotate modes for added challenge

Shuffle and instant Solution options

Random image generator

Smooth zoom controls

Clean and minimal interface

Your goal is simple: reconstruct the complete image by correctly positioning each tile. Sounds easy? Try it and test your brainpower!

Perfect for players who enjoy strategic thinking and visual problem-solving.
